## 🗂️ โครงสร้างข้อมูล

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| `ID_CONSTR` | รหัสประเภทอาคาร (เช่น `105`, `506/1`) |
| `NAME_CONSTR` | ชื่อประเภทอาคาร (เช่น บ้านพักอาศัยตึกสองชั้น) |
| `CHANGWAT_CODE` / `CHANGWAT_NAME` | รหัส / ชื่อจังหวัด |
| `PRICE_CONSTR` | ราคาค่าก่อสร้าง (บาท/ตารางเมตร) |

**6 หมวดอาคาร** (จากหลักแรกของ `ID_CONSTR`): `1xx` บ้านพักอาศัย · `2xx` บ้านแถว/ทาวน์เฮาส์ · `3xx` ห้องแถวไม้ · `4xx` ตึกแถว · `5xx` อาคารเฉพาะกิจ (โรงแรม/โรงพยาบาล/โรงงาน…) · `6xx` งานภายนอก (รั้ว/ถนน/สระว่ายน้ำ…)

## 📊 สิ่งที่อยู่ใน Notebook

1. **Treemap** — หมวดอาคาร → ประเภท (ขนาด/สี = ราคา)
2. **แผนที่ Choropleth** — ราคาค่าก่อสร้างเฉลี่ยรายจังหวัด
3. **Box plot** — การกระจายราคาตาม 6 หมวด
4. **Top 15** ประเภทอาคารแพงสุด
5. **K-Means (k=4)** + PCA — จัดกลุ่มจังหวัดตามโครงสร้างราคา
6. **🔮 Ridge Regression** — ประเมินราคา (ด้านล่าง)

## 🔮 Predictive Model — Ridge Regression

ทำนาย*ราคาค่าก่อสร้าง (บาท/ตร.ม.)* จาก **ประเภทอาคาร (69) + ภูมิภาค (6)**

- **ประเมิน:** 5-fold cross-validation → **R² 0.988 · MAE 227 ฿/ตร.ม.**
- ประเภทอาคารเป็นตัวกำหนดราคาหลัก · ภูมิภาคปรับตามต้นทุนพื้นที่ (แม่ฮ่องสอน/ภาคใต้แพงกว่าเพราะค่าขนส่ง)
